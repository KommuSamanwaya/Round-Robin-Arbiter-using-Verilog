# Round-Robin-Arbiter-using-Verilog
4-Request Round Robin Arbiter implemented in Verilog HDL with RTL design, functional verification, simulation, and logic synthesis.

Overview:
This project implements a 4-Request Round Robin Arbiter in Verilog HDL. A Round Robin Arbiter is a hardware arbitration mechanism that provides fair access to a shared resource by rotating the priority among multiple requesters after each successful grant, thereby preventing starvation.

The design follows an RTL-based approach and includes functional verification using a dedicated Verilog testbench. The project also includes RTL elaboration, logic synthesis, and schematic analysis to demonstrate the complete digital design flow.

Features:

4-Request Round Robin Arbitration

Fair and Starvation-Free Resource Allocation

Rotating Priority Pointer Mechanism

Verilog RTL Implementation

Functional Verification using Testbench

Simulation Waveform Analysis

RTL Elaboration

Logic Synthesis

RTL/Gate-Level Schematic Analysis
